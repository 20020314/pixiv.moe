## Online Site
[https://pixiv.moe](https://pixiv.moe)

## Dev
```bash
# Start for development
git clone https://github.com/kokororin/pixiv.moe
cd pixiv.moe
npm install
npm start
```

### Commands
- Init: `$ npm install`
- Run: `$ npm start`
- Build: `$ npm run dist`
- Publish `$ npm run deploy`
